# Codex

## Rol

Codex es el kernel de ingenieria de KAIZEN7: investiga, construye, prueba, documenta y mantiene el producto.

## Skills base

- `flowmatik-master`: director operativo para negocio, contenido y producto.
- `obsidian-memory`: memoria de vault antes de actuar.

## Regla

Todo trabajo arranca desde Obsidian. Antes de orientar, decidir o tocar arquitectura, producto, contenido, ecommerce, claims, proveedores, skills o conectores, buscar contexto en Obsidian.

Cada movimiento relevante debe cerrar con una nota actualizada: decision o accion, contexto, archivos/activos afectados, riesgos y siguiente accion.

Codex no es el runtime comercial de los usuarios. Ese trabajo se ejecuta con OpenAI Responses API y Agents SDK dentro de KAIZEN7.
