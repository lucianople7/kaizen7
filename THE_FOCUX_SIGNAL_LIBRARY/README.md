# THE FOCUX Signal Library

Storage pesado local en OneDrive para THE FOCUX.

Obsidian y KAIZEN7 son cerebro limpio: criterio, resumen, decisiones y acciones.
Esta carpeta guarda materia prima: PDFs, capturas, imagenes, videos, podcasts, precios, formulas, proveedores, COA y descartes.

## Carpetas

- `00_Inbox` - entrada rapida sin clasificar.
- `01_Fuentes_Reales` - estudios, documentos oficiales, fichas tecnicas, etiquetas y fuentes verificadas.
- `02_Competencia_Formulas_Precios` - capturas, paginas, precios, ingredientes, claims y comparativas.
- `03_Media_Fotos_Videos_Podcasts` - referencias visuales, thumbnails, videos, podcasts y assets para contenido.
- `04_Proveedores_Docs` - proveedores, COA, certificados, especificaciones y conversaciones tecnicas.
- `99_Descartes_Claims_Riesgo` - material dudoso, claims bloqueados, riesgos legales y cosas que no debe copiar THE FOCUX.

## Regla

```text
Archivo pesado aqui.
Resumen y decision en Obsidian.
URL de fuente siempre en Signal Bowl.
```
