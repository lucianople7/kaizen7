# Product Genome

## Decision

La ventaja competitiva de KAIZEN7 no es el proveedor del modelo ni un framework. Es el conocimiento acumulado y evaluado sobre productos reales.

## Flywheel

```text
Producto
  -> Research
  -> Hipotesis
  -> Builder
  -> Campana / tienda / proveedor
  -> Metricas y resultados
  -> Evals y guardrails
  -> Memory
  -> Product Genome
  -> Mejores decisiones
```

## Admission Rules

El Genome no debe guardar cualquier output generado.

Guardar solo si:

- tiene fuente o resultado,
- incluye fecha y contexto,
- supera evaluacion,
- no contradice una decision vigente sin marcarlo,
- y tiene nivel de confianza.
