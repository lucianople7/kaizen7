# K7 Hive Memory Engine

## Decision 2026-06-25

KAIZEN7 incorpora `K7 Hive Memory Engine` como capa operativa para usar memoria colmena con menos pasos y menos tokens.

Formula:

```text
Headroom + Graphify + Ponytail + Obsidian²
```

## Significado

- **Headroom:** comprimir ruido antes de quemar contexto.
- **Graphify:** pensar en nodos y vecinos antes de leer carpetas completas.
- **Ponytail:** accion minima que reduce friccion real.
- **Obsidian²:** memoria canonica + memoria operativa conectada.

## Loop

```text
intencion -> nodo canonico -> vecinos utiles -> decision -> accion minima -> memoria util
```

## Celdas de la colmena

- `Canon`: arquitectura, reglas y decisiones firmes.
- `Daily`: movimientos recientes y bloqueos.
- `Domain`: THE FOCUX, Flowmatik, ecommerce, contenido, proveedores y compliance.
- `Genome`: datos estructurados de producto y negocio.
- `Evolution`: repos, patrones, herramientas y descartes.
- `Skills`: aprendizajes convertidos en accion.

## Regla principal

```text
Leer solo lo que cambia la decision.
Escribir solo lo que evita repetir trabajo.
Convertir lo repetido en skill o SOP.
Descartar lo que no aporta.
```

## No es

- No es una base de datos nueva.
- No instala Graphify ni Headroom por defecto.
- No sustituye Obsidian.
- No aumenta pasos si no reduce tokens o errores.
