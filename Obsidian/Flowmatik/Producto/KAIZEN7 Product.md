# KAIZEN7 Product

Fecha: 2026-06-27

## Definicion madre

KAIZEN7 es el cerebro operativo que ayuda a que cada proyecto llegue a su objetivo con menos pasos, menos tokens y mas criterio.

## Categoria

KAIZEN7 es un **Agentic Product Growth OS**: un sistema operativo de criterio, memoria y ejecucion para proyectos.

No compite como otro chatbot. Compite como el cerebro que coordina agentes, herramientas, memoria, decisiones y verificacion.

## Producto en una frase

KAIZEN7 convierte cualquier objetivo de proyecto en la siguiente accion correcta, usando la minima memoria necesaria, las herramientas adecuadas y una verificacion clara.

## Cliente ideal

- Fundadores con varios frentes abiertos.
- Creadores que quieren convertir una marca en sistema.
- Equipos pequenos que pierden contexto entre herramientas.
- Builders que usan IA, pero necesitan menos ruido y mas direccion.
- Agencias que gestionan varios proyectos y necesitan memoria reutilizable.

## Problema

Los proyectos no fallan solo por falta de herramientas.

Fallan porque:

- se pierde contexto,
- se repiten decisiones,
- se gasta demasiada energia explicando lo mismo,
- cada agente empieza desde cero,
- no hay memoria viva,
- no existe una siguiente accion priorizada,
- se confunde movimiento con avance,
- se publican cosas sin verificacion.

## Promesa

Con KAIZEN7, cada movimiento importante queda convertido en:

- memoria util,
- decision trazable,
- accion minima,
- criterio reutilizable,
- riesgo visible,
- aprendizaje acumulado.

## Objetos del producto

- **Proyecto**: marca, producto, repo, web, campana o sistema.
- **Objetivo**: resultado concreto que se quiere alcanzar.
- **Memoria**: contexto vivo en Obsidian, repo y datos compactos.
- **Senal**: fuente externa que puede mejorar el proyecto.
- **Decision**: criterio aceptado, con fecha y razon.
- **Accion**: movimiento pequeno con impacto real.
- **Verificacion**: prueba de que la accion funciono.
- **Aprendizaje**: regla nueva que reduce pasos futuros.
- **Adapter**: herramienta externa que KAIZEN7 usa sin volverse dependiente de ella.

## Modulos MVP

### 1. Obsidian Memory

Segundo cerebro operativo. Guarda decisiones, criterios, riesgos y siguientes pasos.

### 2. Agent Runner

Entrada simple: objetivo dentro, accion priorizada fuera.

### 3. Agent Advisor

KAIZEN7 aconseja a otros agentes que leer, que evitar, que herramientas cargar y que hacer primero.

### 4. Semantic Memory

Indice compacto de memoria para no leer todo cada vez.

### 5. Signal Radar

Ingesta fuentes de GitHub, Hugging Face, papers, webs, tendencias y herramientas.

### 6. Hunter

Convierte senales en cola de implementacion y oportunidades.

### 7. Smart Crons

Rutinas de revision, radar, limpieza, checks y evolucion.

### 8. Product Genome

Ficha estructurada de cada producto: audiencia, promesa, canales, riesgos, monetizacion y roadmap.

### 9. K7 AI Booster

Capa IA-para-IA que potencia a Codex y otros agentes con memoria, criterio, permisos, herramientas recomendadas, accion minima y verificacion.

## Flujo de uso

1. El usuario declara un objetivo.
2. KAIZEN7 lee solo la memoria necesaria.
3. Recomienda el minimo contexto y skills.
4. Define una accion prioritaria.
5. Ejecuta o guia la ejecucion.
6. Verifica.
7. Escribe aprendizaje en Obsidian.
8. La siguiente vez necesita menos pasos.

Es alguien que ya usa IA, tiene vision, pero necesita un cerebro operativo que mantenga direccion, memoria y criterio.

THE FOCUX funciona como primer caso real, pero no define el limite del producto.

```text
KAIZEN7 = agente / sistema operativo
```

## Diferenciacion

- Memoria primero, no chat primero.
- Adaptadores, no dependencia de una sola herramienta.
- Criterio antes que automatizacion.
- Menos tokens como ventaja operativa.
- Cada decision queda trazada.
- Cada proyecto conserva identidad propia.
- Lo aprendido en un proyecto mejora los siguientes.

## Lo que KAIZEN7 no es

- No es THE FOCUX.
- No es un gestor de tareas generico.
- No es un chatbot decorativo.
- No es un CRM.
- No es una promesa magica de automatizacion total.
- No sustituye el criterio humano: lo organiza, lo recuerda y lo afila.
