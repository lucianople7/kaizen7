# Vault Structure

Recommended structure:

```text
Obsidian/
├── Flowmatik/
│   ├── Arquitectura/
│   ├── Agentes/
│   ├── SOPs/
│   └── Roadmap/
├── TheFocux/
│   ├── Formulas/
│   ├── Proveedores/
│   ├── Branding/
│   └── Regulacion/
├── Research/
├── Ideas/
└── Diario/
```

## Note Types

- Decision notes: final choices, why, date, tradeoffs.
- Research notes: source-backed findings, links, uncertainty.
- SOPs: repeatable processes with inputs, steps, output and quality checks.
- Ideas: raw concepts not yet accepted.
- Diario: chronological log, useful for handoff and retrospectives.

## File Naming

Use stable, readable names:

- `Posicionamiento.md`
- `Formula KETO PERFORMANCE.md`
- `Proveedores Europa.md`
- `Roadmap Q1.md`
- `SOP Crear Content Pack.md`
