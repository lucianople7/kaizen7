# project.connectivity_os

## Purpose

Improve each KAIZEN7 ecosystem project independently while connecting them through small shared contracts.

KAIZEN7 is the coordination layer between humans, AI agents and digital projects. It should connect projects without absorbing their responsibilities.

## Use When

- A mission affects multiple projects.
- Work touches KAIZEN7, Flowmatik, THE FOCUX, Mr. Kaizen or NEUROCITY boundaries.

## Outputs

- project boundary
- shared contracts
- next project action
- writeback target
- kernel impact check

## Kernel Impact Check

Before changing the Kernel, confirm the change improves at least one of:

- complexity
- context
- tokens
- decision time
- execution quality

## Verification

- Boundary is present.
- Shared contract is present.
- Next action is present.
- Project roles are not mixed.
- Kernel impact is explicit.
