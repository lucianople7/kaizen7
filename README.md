# KAIZEN7

KAIZEN7 es el agente y sistema operativo de crecimiento.

Su funcion es dirigir proyectos, conservar memoria, mejorar decisiones, reducir pasos y ayudar a que cada proyecto llegue a su objetivo.

## Relacion con THE FOCUX

KAIZEN7 no es THE FOCUX.

- `kaizen7` es el agente: memoria, metaskills, Obsidian, smart-crons, Signal Radar, decision engine y arquitectura.
- `thefocux` es un proyecto gestionado por KAIZEN7: web publica, marca, Cloudflare, AI layer, MCP publico, contenido, ecommerce y NEUROCITY.

## Regla operativa

1. Las decisiones empiezan en Obsidian.
2. KAIZEN7 decide el minimo movimiento util.
3. Los proyectos se mantienen separados cuando tienen vida publica propia.
4. La memoria interna no se mezcla con repos publicos de marca.
