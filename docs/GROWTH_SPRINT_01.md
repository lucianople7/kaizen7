# KAIZEN7 Growth Sprint 01

## Objective

Turn the KAIZEN7 Kernel into visible growth for:

- THE FOCUX
- Flowmatik
- Kaizen
- KAIZEN7 execution quality

## CEO Filter

Every mission must answer:

```text
What can we launch today that grows the ecosystem?
```

If a mission does not produce an asset, content, decision, lead, product, time saved or friction reduced, it does not enter the sprint.

## Fastest Start

```powershell
npm.cmd run k7:now
```

This command selects the current highest-impact mission and turns it into a compact action card.

## Sprint Mix

- 3 missions for THE FOCUX
- 3 missions for Flowmatik
- 2 missions for Kaizen
- 2 missions for KAIZEN7
