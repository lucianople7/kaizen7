# KAIZEN7 Growth Protocol

## Mission

KAIZEN7 does not exist to build more KAIZEN7.

KAIZEN7 exists to grow projects.

Every day, at least one of these projects should move forward:

- THE FOCUX
- Flowmatik
- Kaizen

KAIZEN7 only improves when that improvement directly helps one of them.

## Main Question

Before starting any mission, ask:

```text
What can we launch today that grows the ecosystem?
```

If the mission does not answer that question, do not execute it.

## Valid Mission Outputs

A mission only matters if it produces at least one of:

- published content,
- reusable asset,
- clear system improvement,
- validated strategic decision,
- lead,
- product,
- sale,
- time saved,
- friction reduced.

If it produces none of these, discard the mission.

## Mandatory Flow

```text
Idea
-> Growth Gate
-> K7 Mission
-> Mission Control
-> Mission Brief
-> Codex/Agent execution
-> Mission Outcome Receipt
-> Next action
```

No exceptions.

## Growth Gate

Mission Control must classify every mission before execution:

- growth lane,
- valid output,
- execute or rewrite.

If the lane is unclear or the output is unclear, the mission is rewritten before any agent receives it.

## Mission Closeout

At the end of every mission, answer:

- What was created?
- What was tested?
- What was learned?
- What is reusable?
- What do we do next?

## When To Improve KAIZEN7

Improve KAIZEN7 only when the improvement does at least one of:

- reduce context,
- reduce friction,
- improve verification,
- save time,
- accelerate future missions.

If it does not do one of these, it does not belong in the Kernel.

## What We Do Not Do

- We do not build features because they are interesting.
- We do not research without an output.
- We do not add complexity without a real need.
- We do not perfect the Kernel while the projects are not growing.

## Success Metrics

Success is not measured by code written, documentation created or capabilities added.

Success is measured by:

- more content,
- more assets,
- more useful automation,
- more sales,
- more growth,
- less time,
- less friction.

## Philosophy

```text
Human decides.
KAIZEN7 coordinates.
Agents execute.
Projects grow.
```

## Anti-Distraction Rule

No theory without receipt.

No decisions without output.

No research without artifact.

No architecture without real use.
