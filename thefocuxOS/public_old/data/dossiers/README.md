# THE FOCUX Public Dossiers

This folder contains public-ready dossier data for THE FOCUX.

Current rule:

```text
Draft dossiers can be exposed as transparent work-in-progress context, but they must not be treated as published medical, product or performance claims.
```

Files:

- `index.json` - public dossier index.
- `energia-con-criterio-draft.json` - first OS Lite dossier draft.

Publication gate:

- `sources` must include external URLs before public publication.
- `blocked_claims` must be reviewed.
- `review_required` must remain `true` until human approval.
- `publish_status` stays `draft` until review.
