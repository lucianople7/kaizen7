# research.pattern_intake

## Purpose

Turn external repositories, models, tools or pasted technical claims into sourceable KAIZEN7 adaptation decisions.

## Use When

- A mission asks to evaluate GitHub, Hugging Face, Pinokio, MCP, CLI tools or architecture patterns.

## Outputs

- source summary
- pattern extracted
- verdict
- risks
- next action

## Verdicts

- `adopt_now`
- `adapt_pattern`
- `test_later`
- `reference_only`
- `reject`
