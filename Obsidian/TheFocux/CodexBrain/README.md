## Limites sanos

Libre para pensar y mejorar. No libre para guardar secretos.

No guardar aqui:

- tokens,
- credenciales,
- datos personales,
- contratos sensibles,
- informacion medica individual,
- claves privadas,
- contenido que deba vivir en compliance, Product DNA o archivos legales.
